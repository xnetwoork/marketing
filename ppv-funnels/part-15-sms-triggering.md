# Part 15: SMS Triggering

## Learning Objectives
- Use SMS for PPV events without over-notifying or increasing opt-outs
- Identify the highest-leverage SMS triggers in a time-bound funnel
- Create SMS copy that is brief, actionable, and trust-preserving

---

## SMS in PPV: High Impact, High Risk

SMS can dramatically increase attendance and last-minute conversions, but it is also the fastest way to damage trust if abused. Treat SMS as **event-critical messaging**, not a general marketing channel.

---

## The 3 Essential SMS Triggers

1. **Event start reminder** (30–60 minutes before)
2. **Access link delivery** (immediately after purchase)
3. **Last call reminder** (2–4 hours before start)

Optional: replay expiration reminder if it’s a real policy.

---

## SMS Copy Structure

Keep messages under 160 characters when possible:
- Event name + time
- Access link or CTA
- Support option (“Reply HELP for support”)

Example structure: “Starts in 1 hour: [Event Name]. Access here: [short link]. Need help? Reply HELP.”

---

## Key Takeaways
- SMS should be used only for event-critical triggers
- Fewer messages with clear actions outperform frequent reminders
- Include support options to reduce anxiety and refunds
- Respecting SMS trust preserves long-term audience value

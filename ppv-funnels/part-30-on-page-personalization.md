# Part 30: On-Page Personalization

## Learning Objectives
- Personalize PPV landing pages by source, location, or audience segment
- Increase conversion without creating fragmented message variants
- Use lightweight personalization tactics that are easy to maintain

---

## Why Personalization Works in PPV

PPV buyers care about **timing, relevance, and clarity**. Personalization helps them instantly see that the event fits their context. The goal is not deep personalization; it’s **immediate relevance cues**.

---

## High-Impact Personalization Elements

- **Local time conversion** for event start
- **Source-specific headline** (e.g., “Welcome MMA fans” for sports channels)
- **Device messaging** (mobile vs. desktop instructions)
- **Geographic availability** (region-specific access policies)

These changes can be implemented without rewriting full pages.

---

## Personalization Rules

- Keep the primary promise consistent across variants
- Limit to 2–3 dynamic fields per page
- Always provide a default fallback for missing data

Personalization should reduce friction, not introduce complexity.

---

## Key Takeaways
- Lightweight personalization increases relevance and conversion
- Time and location cues are the highest-impact PPV personalization elements
- Maintain a consistent value core across all variants
- Over-personalization increases maintenance cost without measurable gain
